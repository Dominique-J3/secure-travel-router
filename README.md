# Secure Raspberry Pi Travel Router

## Overview
This project involves building a secure travel router using Raspberry Pi to protect users on public networks.

## Objectives
- Secure internet traffic using VPN encryption
- Provide network control and monitoring features
- Improve safety on public Wi-Fi networks

## Technologies Used
- Raspberry Pi
- OpenWRT
- VPN configuration (OpenVPN)
- Linux networking

## Features
- Encrypted internet traffic through VPN
- Website blocking (content filtering)
- Device monitoring and network control
- Web-based configuration interface

## Results
- Successfully secured internet traffic in public environments
- Improved user control over network activity
- Demonstrated practical network security implementation

## Skills Demonstrated
- Network security
- System configuration
- Linux administration
- VPN implementation
